Uma Roulette

Sorteos con carreras de caballos animadas · Inspirado en Uma Musume

JavaScript HTML5 CSS3 Bootstrap
Probar en vivo Cómo funciona Código

¿Qué es esto?

Uma Roulette es básicamente una forma divertida de hacer sorteos. Está inspirada en el juego Uma Musume (de ahí los caballos), pero la idea surgió cuando necesitaba una manera más entretenida de elegir ganadores en streams o eventos con amigos.

Lo bueno es que puedes agregar hasta 8 participantes, cada uno tiene su propio caballo animado, y las carreras son 100% aleatorias. No hay trampa ni cartón.

Lo que hace

Gestión dinámica

Agrega o quita participantes cuando quieras, hasta 8 máximo

Caballos únicos

Cada corredor tiene un sprite diferente, todos animados

La parte técnica

Cosas interesantes que resolví

Sistema de sprites únicos

Tenía que asegurarme de que cada participante tuviera un caballo diferente. Creé un "pool" de sprites disponibles que se van asignando y reciclando cuando alguien se elimina. 

function obtenerSpriteUnico() {
    if (spritesRestantes.length === 0) {
        spritesRestantes = [...spritesDisponibles]
            .sort(() => Math.random() - 0.5);
    }
    return spritesRestantes.shift();
}
Animación de las carreras

Usé setInterval para actualizar las posiciones cada 100ms. Cada caballo avanza una cantidad aleatoria, y cuando uno llega al 95% de la pista, se detiene todo y se anuncia el ganador. 

intervaloCarrera = setInterval(() => {
    participantes.forEach((p, i) => {
        p.progreso += Math.random() * 2.5 + 1;
        
        const porcentaje = Math.min(p.progreso, 95);
        elementosCaballos[i].style.left = porcentaje + '%';

        if (p.progreso >= 95) {
            clearInterval(intervaloCarrera);
            mostrarGanador();
        }
    });
}, 100);
Sin frameworks

Todo está hecho con JavaScript vanilla. Manipulación directa del DOM, gestión de estado manual, y eventos puros. A veces es bueno hacer las cosas desde cero para entender bien cómo funcionan.
