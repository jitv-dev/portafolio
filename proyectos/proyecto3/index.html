Sistema de Estacionamiento

Aplicación de escritorio para gestión completa de estacionamientos

Python Tkinter SQLite GUI
Ver características

¿Qué es?

Un sistema completo de gestión de estacionamientos desarrollado en Python. La idea surgió cuando mi tio empezó a trabajar como encargado de estacionamiento, asi que al verlo tan complicado manejando un excel o papel y calculadora para llevar el control, decidí crear algo simple pero funcional.

El sistema permite registrar entradas y salidas de vehículos, calcular costos automáticamente, manejar pagos en efectivo o tarjeta, y llevar un historial completo. Todo guardado en una base de datos SQLite local.

Flujo de trabajo

1. Registro de entrada

Se ingresa la patente y el sistema guarda automáticamente la hora

2. Consultar costo

Calcula el costo actual y congela el tiempo para el pago

3. Registro de salida

Procesa el pago y genera el ticket con todos los detalles

4. Historial

Búsqueda por fecha y visualización de todas las transacciones

La parte técnica

Decisiones técnicas importantes

Tkinter para la interfaz

Elegí Tkinter porque viene incluido con Python, lo que hace que la aplicación sea fácil de distribuir sin dependencias extra. Aunque no es la librería más bonita, es perfectamente funcional para aplicaciones de escritorio simples. 

# Configuración básica de la ventana
root = tk.Tk()
root.title("Sistema de Estacionamiento")
root.geometry("1100x700")
root.configure(bg='#f0f0f0')
SQLite como base de datos

SQLite es perfecto para este proyecto: no necesita instalación ni configuración, todo se guarda en un archivo local, y es súper rápido para el volumen de datos que maneja un estacionamiento pequeño. 

def init_db():
    conn = sqlite3.connect("parking.db")
    cursor = conn.cursor()
    cursor.execute("""
        CREATE TABLE IF NOT EXISTS parking (
            id INTEGER PRIMARY KEY AUTOINCREMENT,
            plate TEXT UNIQUE,
            entry_time TEXT
        )
    """)
    conn.commit()
    conn.close()

Cálculo de vuelto automático

Para pagos en efectivo, el sistema calcula automáticamente el vuelto y lo muestra en un popup grande y claro. Esto acelera el proceso y evita errores de cálculo manual.
